Supervision Zabbix

Déploiement d'un serveur de supervision Zabbix sous Linux Debian pour surveiller proactivement l'infrastructure. Ce projet inclut le déploiement d'Agents sur les serveurs Windows, la configuration SNMP pour les équipements réseaux (Stormshield) et la création de tableaux de bord d'alertes.
