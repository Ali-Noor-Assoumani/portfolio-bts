Plateforme de Ticketing osTicket

Déploiement complet d'une solution de gestion des incidents et des demandes (Ticketing) basée sur osTicket dans un environnement Linux Debian. Ce projet inclut la configuration ITIL (SLA, Catégories), l'authentification LDAP et la gestion du cycle de vie des tickets.

Architecture Réseau

Flux logiques : Le Serveur Web osTicket (sur Debian 11 - 172.16.5.2) est hébergé en DMZ derrière le pare-feu Stormshield. Il communique en LDAP avec le contrôleur de domaine Windows Server (Réseau IN - 192.168.5.100) pour l'authentification.
